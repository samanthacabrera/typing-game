### WPM Typing Test

A fast and minimal typing test featuring:

- Live WPM indicator
- Accuracy tracking
- Wuthering Heights text generator

---

Click the link below to try the live version:

[Typing Test](https://samanthacabrera.github.io/typing-test/)

or run locally:

```bash
git clone https://github.com/samanthacabrera/typing-test.git
cd typing-test
npm install
npm run dev
```
